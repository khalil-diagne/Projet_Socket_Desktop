# Projet Socket Desktop

## 📋 Description

Projet Socket Desktop est une application Python conçue pour les communications réseau basées sur les sockets. Ce projet permet de développer et de tester des applications client-serveur utilisant les sockets TCP/UDP.

## 🎯 Objectifs

- Implémenter une architecture client-serveur robuste  
- Gérer les connexions réseau de manière efficace  
- Fournir une interface bureau pour les opérations réseau  
- Faciliter le debugging et le monitoring des connexions

## 🛠️ Technologies

- **Langage** : Python  
- **Type** : Application Desktop  
- **Architecture** : Client-Serveur avec Sockets

## 📦 Installation

### Prérequis

- Python 3.8 ou supérieur  
- pip (gestionnaire de paquets Python)

### Étapes

```bash
# Cloner le dépôt  
git clone https://github.com/khalil-diagne/Projet_Socket_Desktop.git

# Accéder au répertoire  
cd Projet_Socket_Desktop

# Installer les dépendances (si applicable)  
pip install -r requirements.txt
```

## 🚀 Utilisation

```bash
# Lancer l'application  
python main.py
```

> Veuillez consulter la documentation du projet pour les détails spécifiques d'utilisation.
